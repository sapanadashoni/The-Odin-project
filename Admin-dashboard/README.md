#### Overview
This project is part of The Odin Project curriculum, focusing on practicing and mastering Flexbox and CSS Grid layouts.

#### Purpose
The main goal of this exercise is to:

> Understand and implement Flexbox for flexible and responsive layouts
> Learn CSS Grid for complex two-dimensional layouts
> Apply best practices in CSS layout design
> Build a functional dashboard layout
### What's Inside
index.html - HTML structure for the dashboard
style.css - CSS styling with Flexbox and Grid implementations
### Learning Outcomes
Through this exercise, I practiced:

> Creating responsive layouts with CSS Grid
> Using Flexbox for component alignment and distribution
> Managing space distribution with justify-content and align-items
> Building a sidebar and main content area with proper spacing
